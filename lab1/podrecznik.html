Rys. 4 Program obliczający sumę dwóch liczb całkowitych wprowadzonych z klawiatury i wyświetlający wynik na ekranie - ćwiczenie 3.

Ćwiczenie 3. Deklarujemy zmienne w programie i wykonujemy na nich obliczenia

  1. Program pokazywany na rysunku 4. pozwala na wyświetlenie dwóch liczb całkowitych na ekran. Przepisz kod programu pokazanego na rysunku 4.
  2. Zapisz program w pliku pod nazwą suma. Skompiluj i uruchom program.

Ćwiczenie 4. Modyfikujemy program

  1. Zodyfikuj program Suma zapisany w ćwiczeniu 3. tak, aby wynik działania programu wyglądał jak na rysunku 5.
  2. Zapisz program w pliku pod tą samą nazwą. Skompiluj i uruchom program.

Rys. 5 Wynik działania programu - ćwiczenie 4.
